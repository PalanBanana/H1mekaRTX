# H1mekaRTX v0.3.0 Stage 2 Summary

This milestone summarizes Stage 2 signed activation preparation work.

- Apple entitlement request template
- Apple entitlement request packet
- Apple Developer Portal checklist

- SystemExtensions activation status matrix

## RTX 5070 PCI Target

- Vendor ID: 0x10de
- Device ID: 0x2f04
- IOPCIMatch: 0x2f0410de
- Subsystem Vendor ID: 0x1458
- Subsystem ID: 0x417e

## Current Boundary

The project remains documentation, diagnostics, and read-only DriverKit PCI probing focused.

No real signed DriverKit activation has been attempted yet.

## Not In Scope

- Metal acceleration
- framebuffer support
- GSP initialization
- display engine initialization
- MMIO writes
- BAR memory poking
- GPU reset logic
- general RTX 5070 macOS acceleration

## Next Stage

The next stage should focus on Apple entitlement submission, Team ID and bundle ID confirmation, provisioning profile setup, and signed activation only after entitlement approval.
