# H1mekaRTX v0.2.0 Stage 1 Summary

## Scope

This milestone summarizes the current Stage 1 state of H1mekaRTX.

## Completed

- macOS Sonoma installation documented
- RTX 5070 PCI visibility documented
- Xcode and DriverKit environment documented
- DriverKit project skeleton added
- RTX 5070 read-only PCI match added
- Read-only PCI config-space probe added
- Host App skeleton added
- SystemExtensions activation request plan documented
- Host App activation request UI added
- Entitlement placeholder plan documented
- Bundle ID activation map documented
- SystemExtensions diagnostics script added
- Signing readiness checklist documented
- Signing readiness audit script added

## RTX 5070 PCI Target

- Vendor ID: 0x10de
- Device ID: 0x2f04
- IOPCIMatch: 0x2f0410de

## Current Boundary

The project is still read-only and planning-focused.

No real DriverKit activation has been attempted yet.

## Not In Scope

- Metal acceleration
- framebuffer support
- GSP initialization
- display engine initialization
- MMIO writes
- BAR memory poking
- GPU reset logic
- general RTX 5070 macOS acceleration

## Next Stage

The next stage should focus on signed activation preparation only after valid Apple Developer signing, provisioning profiles, DriverKit entitlement approval, and PCI transport entitlement configuration are available.
