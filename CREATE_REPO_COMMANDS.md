# Create and push this repo

## Browser method

1. GitHub → New repository
2. Repository name: `H1mekaRTX`
3. Visibility: Public or Private
4. Do not initialize with README if pushing this local folder as-is
5. Create repository
6. Run the commands below

## Command line

```bash
cd H1mekaRTX
git init
git branch -M main
git add .
git commit -m "Initial H1mekaRTX research scaffold"
git remote add origin https://github.com/<YOUR_USERNAME>/H1mekaRTX.git
git push -u origin main
```
